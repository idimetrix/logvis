## logvis

#### TypeScript/JavaScript promise queue client library with concurrency control

## Installation

Install with [npm](https://www.npmjs.com):

```sh
$ npm install --save logvis
```

Install with [yarn](https://yarnpkg.com):

```sh
$ yarn add logvis
```
